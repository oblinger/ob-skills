## What it is

The **Glossary** facet marks a `## Glossary` H2 a document may carry — an alphabetically-ordered list of `**Term** — definition` lines. It is a *doc facet* (an authored content region), not a structural part of the anchor.

## Rules

- RULE (alpha-order): entries are sorted by term, A–Z — never grouped by category.
- RULE (one-line): each entry is `**Term** — one-sentence definition`; longer treatment links out to the canonical page.

## Where it lives

Inline in the document that needs it, as the last H2 before any `# BRIEF`. A facet is a single spec page — it has no sub-folder and no member zone (hence this page is **masthead-only**, the canonical shape for the *facet* kind of anchor page).

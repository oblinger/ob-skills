# CSE PRD

## Overview

`/cse` is a reference skill that demonstrates the canonical CAB skill-anchor structure. It is **not user-invocable** — its purpose is to be a worked example for skill authors and for the CAB spec to point at.

## Goals

- Show the file shapes of an anchor in CAB skill-trait form.
- Demonstrate where `SKILL.md` lives, where action files live, and where Plan / Features documents live.
- Pair with [[CAE]] to give skill-anchor authors a canonical reference.

## Non-goals

- Doing actual work. CSE has no semantic behavior.
- Being invokable. `/cse demo` exists only as an illustrative action file.

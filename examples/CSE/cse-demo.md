# cse-demo

Illustrative skill action. Demonstrates the kebab-case file-naming convention for skill actions and shows where action runbooks live in a skill anchor.

## Runbook

1. Read the user's argument string.
2. Print `CSE-DEMO: <argument>` followed by a UTC timestamp.

(This action exists to demonstrate the file format. Real skills carry substantive runbooks here — see `~/.claude/skills/feature/feature-create.md` or `~/.claude/skills/audit/audit-rules.md` for working examples.)

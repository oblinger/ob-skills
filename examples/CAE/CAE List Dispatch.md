> **Canonical list dispatch.** A [[Collection]] with **≤ ~15 members** → a **flat member list**: one row per member, the label a link to that member, a one-line description in the cell. No `+` (members are leaves, not containers); no groups (it graduates to [[CAE Grouped Dispatch|member groups]] only past ~15, per [[granularity]]). Hand-ordered (manual). Still **ends with an electric-list marker** (`| ... |`) so a newly-added machine lands there until placed. *(A figure may precede the table per [[CAE Figure Page]]; omitted here because the list is the whole story.)*

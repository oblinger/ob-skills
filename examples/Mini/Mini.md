# Mini
Mini is a deliberately tiny example project whose only purpose is to host the **minimal** instance of each Design-Docs facet (PRD, Architecture, Testing, Decisions, Stories) as one small coherent world — the lean counterpart to the maximal [[HBR]] world.

Its design docs live in `Mini Design/`; each facet lane drops its minimal instance there.

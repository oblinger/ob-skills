## Overview

Mini is a one-command note-stamper: `mini stamp <file>` prepends the current date to a text file. It exists only to host the minimal instance of each Design-Docs facet.

## Goals

- Stamp a file with today's date in one command.

## Non-Goals

- Anything else.

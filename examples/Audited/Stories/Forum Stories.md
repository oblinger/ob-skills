| [[US-FORUM-17 — Add Supporting Evidence\|US-FORUM-17]] | Suggest supporting evidence to substantiate arguments |
| [[US-FORUM-18 — Flag Weak Points\|US-FORUM-18]] | Identify weak points in one's own arguments to address proactively |
| **Cross-Partisan Challenger** |  |
| [[US-FORUM-19 — Rebut Claims\|US-FORUM-19]] | Rebut specific claims in opposing positions |

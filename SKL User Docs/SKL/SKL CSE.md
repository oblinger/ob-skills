# CSE

Reference skill, not invokable.

CSE is the **Common Skill Example** — a worked example of a CAB skill anchor. It pairs with [[CAE]] (the general anchor example) and is referenced by skill authors when they need to see the canonical shape of a skill folder.

There is no real action; CSE exists for structure, not behavior. Navigate to [[CSE]] to see the anchor root page, or [[CSE/SKILL|SKILL.md]] to see the skill spec.

**Find** locates one specific match for given criteria and returns identifier + canonical URL + 1-line context + confidence + sources. It disambiguates when candidates score close, rather than silently picking. For just identifying — not profiling (use [[SKL Describe]]) or comparing many (use [[SKL Survey]]).

Invoke: *"find me X"* / *"find the GitHub repo for X"* / *"find John Smith at Acme as VP Engineering."*

Outputs: [[Lookup]] (`~/ob/kmr/Topic/Search/Lookup/`).

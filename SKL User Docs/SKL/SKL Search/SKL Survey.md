**Survey** builds a multi-dimensional comparison table across many entities + interpretive notes. Two meta-survey forms: **lightweight** (survey-of-surveys to derive dimensions) and **full** `/survey meta <noun>` (three composed surveys plus cell-certainty notation — see [[SKL Meta Survey]]). For comparison — not single-entity profiling ([[SKL Describe]]) or single-instance lookup ([[SKL Find]]).

Invoke standard: *"survey project management SaaS"* / *"compare CRMs for a 20-person team"* / *"what's the landscape of X."* Invoke meta-survey: *"meta-survey feature-flag libraries for Go"* / *"/survey meta software — observability tools."*

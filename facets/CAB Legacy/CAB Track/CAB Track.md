# BRIEF

- **This page is the CAB project's Track umbrella** — the anchor-folder dispatch page for CAB's own project-work surfaces (Backlog, Features, Discussion) plus a small set of agent-facing sidecars (Maintain, Messages). Edits to this page mean editing the dispatch table or the scope note, not piling content here.
- **In-scope rows are CAB *project-work* surfaces only:** Backlog (workflow-state work), Features (F-numbered specs), Discussion (planning trade-offs), and tightly-coupled operational sidecars (Maintain skill, Messages inbox). Anything else lives elsewhere.
- **Out-of-scope: CAB *spec* content.** Traits/, Facets/, Rules/, Aspects.md, Base.md, Conventions/ are externally-consumed reference material that lives at the CAB root per F094 Q6=C — do NOT add rows for them here, and do NOT migrate them under Track.
- **Inclusion test for a new row:** the artifact must be (a) about how CAB-the-project gets built or coordinated, and (b) live inside `CAB Track/`. If both are true, add it; otherwise it belongs at the CAB root or in another anchor.

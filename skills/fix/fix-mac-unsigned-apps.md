# Fix: Open Unsigned Mac Apps

When macOS blocks an app with "can't be opened because it is from an unidentified developer."

## Runbook

### 1. For a standalone .app file

Right-click (or Control-click) the app, then choose **Open** from the context menu. This is different from double-clicking — it gives an option to open anyway.

### 2. For a .dmg with an installer

- Control-right-click the installer package inside the .dmg
- Choose Open — the installer will now show an option to proceed

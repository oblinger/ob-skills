`/create` requires an argument specifying what to create:

- `/create anchor` — new project with folder structure, config, dispatch tables
- `/create feature` — new dated feature design doc
- `/create wp` — new dated work product folder
- `/create spec` — new implementation spec for a milestone
- `/create rule` — new project rule

**Describe** builds a thorough structured description of one entity. Quick is a summary card; Standard is a full profile; Deep ("Dig") is a full dossier with multi-source synthesis, contradictions surfaced, and gaps named. For one entity — not finding one ([[SKL Find]]) or comparing many ([[SKL Survey]]).

Invoke: *"describe Acme Corp"* / *"profile John Smith"* / *"dig into the Pixel 9."*

Outputs: [[Profile]] (`~/ob/kmr/Topic/Search/Profile/`).

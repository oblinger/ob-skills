When the search agent runs against an **organization** — finding, describing, or surveying — these are the rules that apply.

**Public vs private** matters: SEC filings are ground truth for public companies; for private companies the agent triangulates Crunchbase / PitchBook / news / employee proxies and notes ranges rather than point values. Nonprofits have 990s.

**Brand vs legal entity**: by default the agent uses the brand (Google vs Alphabet); it surfaces the legal entity when M&A / contracts / litigation context applies.

Sub-types worth knowing about: startup · public company · nonprofit · government / agency.

When the search agent runs against a **person** — finding, describing, or surveying — these are the rules that apply.

**Privacy is load-bearing.** Public figures are researched as you would any public entity. Private individuals are kept conservative — only what they've themselves published or what's in legitimate news; never PII compilation, never doxxing-shape aggregations. The agent will push back before crossing the line.

Sub-types worth knowing about: public figure · researcher / academic · founder / operator · private individual.

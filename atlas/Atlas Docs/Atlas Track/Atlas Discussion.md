# Atlas Discussion

The home for open-ended trade-off conversations about the Atlas skill. **Decisions don't land here.** When a discussion resolves, the resolution moves to whichever artifact owns the decision:

- `Atlas PRD.md` — product-shape decisions (what the skill is for; what it produces)
- `Atlas Design.md` — architecture decisions (how the skill is structured internally)
- `atlas/SKILL.md` — agent-rule changes (decisive runbook update)
- `SKL Atlas.md` — user-facing guidance changes (matrix, decision shortcuts)

The discussion entry itself stays here as a historical record, with a one-line resolution pointer.
